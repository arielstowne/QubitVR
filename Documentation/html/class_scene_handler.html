Detailed Description

This is used to trigger events associated with the laser pointer.

The laser pointer can be used on objects by either putting if statements in this script that checks e.target.name or tag or by writing a script for the object that implements OnPointerClick, OnPointerEnter, OnPointerExit (see Qubit_Handler for example). Attaches to Player object.
