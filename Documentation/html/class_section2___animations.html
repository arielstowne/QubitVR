Detailed Description

Controls animated gate movements for the section 2 tutorial.

Controls Section 2's tutorial sections in which gates are animated to collide with qubits. Attached to section2_manager->section2_tutorial->Animated Gates->Placeholders->S2, T2. Ensure the Animated Gates object is in the same location as the qubit.
