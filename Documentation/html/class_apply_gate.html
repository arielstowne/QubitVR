Detailed Description

Primary controller for qubit engine.

Controls math and vector rotations and creates and stores History. Stores current qubit state and converts into xyx coord for vector. Stores matrices for gates. Note that this class alerts ModuleManager when a qubit is measured. Attached to qubit shell game object.
